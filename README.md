# Recrut
Recrut – Plateforme de Recrutement
##Contexte du projet

Recrut est une plateforme web développée avec Laravel, dont l’objectif est de connecter les recruteurs (entreprises) et les chercheurs d’emploi selon leurs spécialités.

Le projet respecte les bonnes pratiques de développement et est conçu pour être évolutif et maintenable.

##Objectifs du projet

Utiliser le framework Laravel

Mettre en place une authentification sécurisée

Gérer des profils utilisateurs

Implémenter une recherche simple

Préparer le projet pour des évolutions futures

##Technologies utilisées

Laravel (Framework PHP)

Laravel Breeze ou Jetstream (Authentification)

MySQL (Base de données)

Blade / Tailwind CSS (Interface utilisateur)

JavaScript (Interactions dynamiques)

##Types d’utilisateurs

Recruteur : représente une entreprise

Chercheur d’emploi : possède une spécialité professionnelle

##Fonctionnalités principales
1 - Authentification

Inscription et connexion sécurisées

Gestion des rôles (Recruteur / Chercheur d’emploi)

Déconnexion

Protection des routes selon le type d’utilisateur

2️ - Profil utilisateur

Consultation du profil

Modification des informations :

Nom

Photo de profil

Bio

Spécialité (pour les chercheurs d’emploi)

Changement du mot de passe avec :

Vérification de l’ancien mot de passe

Confirmation du nouveau mot de passe

3️ - Recherche

Recherche d’utilisateurs par :

Nom

Spécialité

Consultation du profil d’un utilisateur trouvé
